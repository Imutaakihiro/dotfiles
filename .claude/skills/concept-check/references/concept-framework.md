# コンセプトメイク フレームワーク

> 出典：Brain「SUPER CONCEPT」（著：パグ）

---

## コンセプトの定義

**「その事業が『誰のどんな問題を解決するのか』を具体的にまとめた構想」**

- 想いや理念とは別物。重要なのは**課題解決の価値提供**。
- コンセプトが曖昧だと、訴求を強くしても・値段を安くしても・ハイパーオファーを作っても滑る。

---

## コンセプトの3階層

| 階層 | 定義 | 特徴 |
|------|------|------|
| ①事業コンセプト | 誰の・どんな問題を・どうやって解決するか | プロフィールにそのまま出る |
| ②商品コンセプト | 事業コンセプトの範囲内で細分化 | 必ず①の中で作ること |
| ③アカウントコンセプト | 顧客が最も関心があり・目新しく・必要だと思う部分に絞る | あれもこれも禁物 |

---

## コンテンツコンセプトの基本4要素

| 要素 | 内容 |
|------|------|
| 誰が | 発信者の属性・実績 |
| 誰に | ターゲット |
| どんな手段で | 手法・アプローチ |
| どんな結果を | 顧客が得るもの |

---

## 良いコンセプトの公式

> **①目新しさ × ②顧客の関心度 × ③信頼**

### ①目新しさ

業界やその顧客にとって新しければOK。新技術・新性能である必要はない。

掛け合わせ例：
- コンテンツ販売専門LINE構築
- 架電を使った高単価セールス
- zoom/LINE診療

### ②顧客の関心度

どれだけ素晴らしいコンセプトでも、顧客の関心がない内容を入れても意味がない。

**関心度ミスマッチ例（NG）：**

| コンセプト | 問題 |
|------------|------|
| 年商数億 | ほとんどの人にとってその段階ではない |
| 初心者から月収200万円 | 初心者には重すぎる |
| Brainアフィリで月収50万円 | その手段で稼ぎたい人がそもそも少ない |
| パリコレモデルのような体型 | ダイエット初心者の理想とはかけ離れすぎ |

**刺さる例：**
- 「自炊めんどくさい」→ 『ずぼらダイエット』
- 「就活でメンタル崩壊」→ 「メンヘラ就活生の内定術」
- 「ポジティブ系で疲れた復縁希望者」→ 「彼をコントロールする復縁術」

### ③信頼

> **信頼と真実性 ＞＞＞＞＞ 興味性**

実績・権威が必ずしも必要なわけではない。主張を裏付ける"何か"があればOK。

**非稼ぐ系の信頼付与例：**
- 「栄養研究歴4年の私が考案した、究極にズボラを追求した〜」
- 「行動心理学専攻の私が教える、〜」
- 「ヲタ歴10年の私が身につけた、〜」

（太字の前文があるかないかで訴求の強さが全く変わる）

## 2つのNG

### NG①：ありきたりにしない

- 「月収100万円稼ぐコンテンツ販売者」
- 「自動化で月100万円」

→ 既視感があり、情報発信の質が高くない限りは避ける

### NG②：ユニークなだけにしない

ユニークさ ≠ フォローする理由。ユニークさがターゲットに魅力的に映るかどうかが問題。

| コンセプト | 評価 |
|------------|------|
| 元自衛隊の副業生活 | ユニークだが刺さる層が限定的 |
| 元自衛官の教える鋼の体の手に入れ方 | 筋トレ×本物の強靭さを求めるターゲットに刺さる |

## 初心者向けコンセプト作成フロー（3ステップ）

```
Step1：絞る（ターゲット or 手段 or 結果）
　↓
Step2：訴求を強くする
　↓
Step3：競合と比較して削るか付け加える
```

### Step1：絞り方3パターン

**パターン①：ターゲットを絞る**

| 軸 | 例 |
|----|-----|
| 職業・職種 | 看護師・保育士・介護士・営業職 |
| 状況 | 転職活動中・夜勤の多い・新生活で一人暮らし |
| 悩み | 恋愛経験が少ない・続けられるか不安 |
| 性格 | ズボラ・完璧主義・3日坊主・不安症 |

コツ：訴求を強くした際に「ある程度の人数がいそうなまとまり」があること。

**パターン②：手段を絞る**

| ジャンル | 手段例 |
|----------|---------|
| ビジネス | アフィリエイト・Brain・SEO・広告・Excel |
| 恋愛 | マッチングアプリ・心理学 |
| 勉強・資格 | 書き込み勉強法・勉強管理術 |

**パターン③：結果を絞る**

| ジャンル | 結果例 |
|----------|---------|
| ビジネス | 月10万円副業で稼げる人生 |
| 恋愛 | マッチングアプリで無双する |
| ダイエット | スラっとしたモデル体型 |

## コンセプトパーツ7選

あなたのコンセプトに1つ入れるだけで際立ちがよくなる。

| # | パーツ | 例 | 向いているジャンル・注意 |
|---|--------|----|--------------------------|
| ① | **手軽感** | 1日3分で・3ステップで・〜するだけ | ダイエット・勉強・スポーツ。投資・副業では怪しくなるので禁物 |
| ② | **意外性** | 炭水化物OKダイエット・化粧品を使わないスキンケア | ジャンルの常識や一般的な考えの否定 |
| ③ | **共感** | 月収17万円だった私が〜・コミュ障だった私が〜 | 特に女性向け |
| ④ | **裏技感** | カンニングペーパーを使い・チート使い放題 | 特に男性向け |
| ⑤ | **具体的な手法** | Lステップを使い・〜式〜法 | ビジネス・投資・勉強・ダイエット |
| ⑥ | **スタート×ゴールのギャップ** | うつ病からナンパ師に・体重90kg→47kg | 人生が変わるレベルの訴求が可能な場合 |
| ⑦ | **ベネフィット重視** | 旅をして自由に生きる・誰もが羨む体型に | 手法・実績での差別化が難しい場合 |

## スーパーコンセプトの作り方（4ステップ）

```
STEP1 競合リサーチ
　↓
STEP2 顧客リサーチ
　↓
STEP3 自分の差別化ポイントを書き出す
　↓
STEP4 コンセプトを削るもしくは追加する
```

### STEP1：競合リサーチ

- SNSなら伸びているアカウントTOP5、SNS以外なら広告・検索上位サイトから調査
- ピックアップ項目：**ターゲット・手法・結果・差別化ポイント**

### STEP2：顧客リサーチ

| 顧客タイプ | 調べること |
|------------|------------|
| 競合の商品・サービスを使っている人 | 悩み・解決できない理由・解決後の理想・顧客の常識 |
| 競合の商品・サービスを使っていない人 | 同上 |

### STEP3：差別化ポイントを書き出す

| キーワード | 例 |
|------------|----|
| 業界初 | 業界初・〜を使ったダイエット |
| 元〜〜 | 元日本代表・元上場企業勤務 |
| 実績・資格・肩書き | 1000人以上サポート・税理士資格保有 |
| 特徴 | 個室自習室・1日10分家庭教師 |
| 性格・キャラ（共感向き） | メンヘラ・ADHD・HSP |
| 最新 | 最新AIの適職診断 |
| 常識の否定 | 努力しない社長 |

### STEP4：コンセプトを削る・追加する

整理テンプレート：

| 分類 | 内容 |
|------|------|
| 誰が | |
| 誰に | |
| 手段 | |
| 結果 | |
| 競合の弱点 | |
| 顧客の悩み | |
| 差別化ポイント | |
| **削る部分** | |
| **追加する部分** | |

## さらにスーパーコンセプトにする掛け合わせパターン

主に商品コンセプトで使う：

| パターン | 例 |
|----------|----|
| ①需要 × 需要 | Twitter × ローンチ（永遠に売れるテーマ同士） |
| ②需要 × 興味性 | 成約率8割超を達成したzoomセールス講座 |
| ③王道 × 別業界 | コンテンツ販売×広告運用 |
| ④需要 × 目新しさ | コンテンツ販売×最新米国式マーケティング |

## コンセプトの段階的な変化

| フェーズ | 推奨するコンセプト |
|----------|--------------------|
| 初期 | 具体的・強い訴求。絞ったコンセプト |
| アカウントパワー増加後 | 抽象化して、より多くの層に刺さるよう変化 |

注意：どこまで抽象化してもコンセプトがイメージできるか？どこまで発信領域を広げるか？の2軸でバランスを取ること。
