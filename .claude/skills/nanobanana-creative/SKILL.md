# nanobanana クリエイティブプロンプト生成スキル

## ワークフロー概要

```
Step 1: ヒアリング
Step 2: デザイン戦略の共有
Step 3: 完全YAMLプロンプト生成（画像生成 + テキスト挿入を1つにまとめて出力）
Step 4: 生成結果のPDCA（画像添付）
```

---

### Step 1: コンセプト設計（ヒアリング）

ユーザーから以下を引き出す。情報が足りない場合は質問する。

- **目的**: 何のために作るか（認知拡大、LP誘導、フォロワー獲得など）
- **媒体**: どこに使うか（インスタフィード・ストーリー・広告バナーなど）
- **ターゲット**: 誰に届けたいか（年代・性別・悩み・欲求）
- **伝えたい1つのメッセージ**: ベネフィットを1つに絞る
- **ブランド/世界観**: トーンや既存のブランドイメージがあれば

### Step 2: デザイン戦略を言語化してユーザーに共有

生成前に以下を簡潔に提示し、方向性を合わせる。

1. **視線誘導**: Z型/F型のどちらを採用し、何をアイキャッチ・CTAに置くか
2. **配色戦略**: ターゲットの感情に合うベースカラー/メインカラー/アクセント（70:25:5）
3. **Vibe（雰囲気）**: 1行で表現できる感情・空気感
4. **4パターンの差別化軸**: 各バリエーションで何を変えるか（例：アングル・スタイル・訴求軸）

詳細なデザイン原則 → `references/design-principles.md`
プロンプトキーワード集 → `references/prompt-techniques.md`

### Step 3: 完全YAMLプロンプト生成（4パターン）

**画像生成パートとテキスト挿入パートを1つのYAMLにまとめて出力する。**
各パターンの差別化ポイントをパターン名に明記すること。

```yaml
# パターンX: [バリエーション名・差別化ポイントを一言で]
企画タイトル: ""
目的: ""
ターゲット: ""

画像仕様:
  アスペクト比: ""   # 1:1（インスタフィード）/ 9:16（ストーリー・リール）/ 16:9（横長バナー）

Vibe:
  感情スタイル: ""

# ── nanobananaに渡す完成プロンプト（画像 + テキスト一括生成）──
ビジュアル:
  メインビジュアル: ""    # 被写体・シーンを具体的に
  アングル構図: ""
  ライティング: ""
  背景: ""
  カラーパレット: ""      # ベース/メイン/アクセントの3色
  スタイル指定: ""

テキストレイアウト:        # nanobananaに画像内に含めるよう指示する
  メインコピー:
    文言: ""              # 例: "デスクを整える、唯一無二の静寂。"
    位置: ""              # 例: 右上エリア
    フォント: ""          # 例: エレガントな明朝体
    色: ""                # 例: ゴールド #C9A96E
  サブコピー:
    文言: ""
    位置: ""
    フォント: ""
    色: ""
  CTA:
    文言: ""              # 例: "詳しく見る ›"
    位置: ""              # 例: 右下
    色: ""

ネガティブプロンプト:
  - ロゴ・透かし
  - 不自然な手・指が多い
  - 低画質・ぼやけた
  - 崩れた文字・謎の言語
  - [ターゲットに合わせて追加]
```

全パターン出力後、末尾に以下を必ず添える：

> **個別コピー:** コピーしたいパターンを教えてください（例：「A」「B」）。
> そのYAMLだけを単独で出力するので `/copy` で取得できます。

ユーザーがパターン名（A/B/C/D）を指定してきたら、**そのYAMLのコードブロックのみ**を返す。説明文は一切添えない。

### Step 4: 生成結果のPDCA（画像添付）

ユーザーが生成した画像を添付してきたら、以下の観点で分析してフィードバックする。

1. **テキストゾーンの再現度**: YAMLで指定した余白・背景色が確保されているか
2. **Vibeの一致**: 意図した雰囲気が出ているか
3. **テキスト挿入可否**: テキストゾーンの背景が複雑すぎないか
4. **技術的な問題**: 不自然な部分・AIアーティファクトがないか

改善点があれば**修正YAMLを1パターンのみ**出力する（テキスト挿入パート含む）。
