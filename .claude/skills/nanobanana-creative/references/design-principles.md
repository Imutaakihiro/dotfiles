# デザインの基礎原則

## デザインの根幹：「誰に」「何を」「どう」

すべてのデザインの出発点。テクニックより先に固める。

- **誰に（ターゲット）**: どんな悩みを持ち、何を求めているか
- **何を（メッセージ）**: ベネフィットを1つに絞る（「1つの発信＝1つのメッセージ」）
- **どう（クリエイティブ）**: フォント・配色・レイアウトで表現する

## デザインの4原則

| 原則 | 意味 | 実践例 |
|------|------|--------|
| **近接** | 関連要素は近づけ、無関係なものは離す | キャッチコピーとサブコピーをまとめる |
| **整列** | 見えないガイドラインに沿って配置 | 左揃え統一でスッキリした印象 |
| **反復** | フォント・色・レイアウトを繰り返す | シリーズ投稿で同じルールを使う |
| **コントラスト** | サイズ・色・太さで対比を作る | 見出しを極端に大きくして視線誘導 |

## レイアウトと視線誘導

**視線パターン（人間が意図して設計すること）:**
- **Z型**: 左上（キャッチ）→ 右上 → 左下 → 右下（CTA）。横長コンテンツに適す
- **F型**: 上部を横に→ 下に流れる縦読み。テキスト主体のコンテンツに適す

**グリッドシステム**: 要素をグリッドに沿って配置することで整理された安定感が生まれる。

## 配色の黄金比

**70:25:5の法則**:
- **ベースカラー（70%）**: 背景・余白。全体のトーンを決める
- **メインカラー（25%）**: 主要テキスト・重要要素
- **アクセントカラー（5%）**: CTA・強調ポイント

**色の心理効果（配色選択の参考）:**
| 色 | 心理的メッセージ | 用途例 |
|----|-----------------|--------|
| 赤 | 情熱・興奮・緊急 | セール・限定・エネルギー系 |
| 青 | 信頼・誠実・冷静 | 金融・医療・BtoB |
| 黄 | 明るさ・注意・希望 | 食品・子供向け・注目 |
| 緑 | 自然・安らぎ・健康 | ヘルスケア・環境・オーガニック |
| 黒 | 高級感・洗練・強さ | ラグジュアリー・モード |
| 白 | 清潔・シンプル・余白 | ミニマル・医療・テック |
| ピンク | 優しさ・女性・愛 | 美容・恋愛・子育て |
| オレンジ | 親しみ・活気・創造 | エンタメ・SNS・フード |

## フォントの選び方

| 書体 | 印象 | 適したシーン |
|------|------|------------|
| 明朝体 | 上品・伝統・信頼 | 高級品・和風・長文 |
| ゴシック体 | 現代的・読みやすい | Web・SNS・見出し |
| 筆書体・手書き風 | 温かみ・親密さ | 和風・手作り感 |
| デザイン書体 | 独自性・個性 | ブランド・アート系 |

## 余白（ホワイトスペース）

- 無駄な空間ではなく、情報を「呼吸」させる重要要素
- ゆったりとした余白 → 上品・洗練された印象
- 情報を詰め込みすぎると「スーパーの特売チラシ」になる

**緩急のバランス**: 一貫性（反復）だけでは飽きられる。時折サプライズ（アクセント・異なる見せ方）を入れて読者の興味を引きつける。

## 写真 vs イラストの使い分け

| 写真 | イラスト |
|------|---------|
| リアリティ・信頼性を伝えたい | 抽象的な概念・図解 |
| 具体的な情報（製品・人物） | 親しみやすい世界観 |
| 感情に訴える（表情・風景） | 写真で表現困難なもの |

## 装飾・あしらい

罫線・アイコン・図形・縁取りなどは「目的」のために使う。

- 情報を区切る
- 特定の要素を目立たせる
- ブランドの世界観を演出する

アイコン・ピクトグラムは言葉を使わずに情報を伝える強力な視覚言語。**やりすぎ禁止。**
