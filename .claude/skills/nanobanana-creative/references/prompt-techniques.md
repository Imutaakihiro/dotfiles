# プロンプトテクニック集

## 基本構造の3要素

```
何が（被写体）＋ どんな状態で（様子・行動）＋ どんな雰囲気で（スタイル・画風）
```

## 構図・カメラワークのキーワード

### アングル
- `ローアングルから見上げた` — 迫力・威圧感・ドラマ性
- `俯瞰（バードアイ）ショット` — 全体感・整理された印象
- `クローズアップ` — 感情・質感・インパクト
- `中景ショット` — バランスのいい自然な印象
- `極端なクローズアップ（マクロ）` — 細部・テクスチャを強調

### 配置・余白
- `中央に配置` — 安定・対称
- `左向きに配置、右側に余白を持たせる` — テキスト追加スペース確保
- `三分割法に基づいて配置` — 自然なバランス

### 写真的な質感
- `被写界深度が浅い（背景がボケている）` — プロっぽさ・主役を際立たせる
- `ゴールデンアワー（日の出・日の入り時の光）` — 温かみ・映画的
- `ドラマチックなライティング` — 緊張感・高級感
- `スタジオライティング` — クリーン・商業的
- `チアロスクーロ（明暗の強いコントラスト）` — 芸術的・重厚感

## Vibe（雰囲気）の表現キーワード

### 感情・空気をそのまま伝える
- 「ちょっと切ないけど、希望が見える雰囲気」
- 「深夜の都会の路地裏みたいな、クールで孤独な雰囲気」
- 「思わず笑っちゃうくらい面白い」
- 「夢の中にいるような幻想的な空気感」
- 「朝の澄んだ空気のような、清潔で前向きな雰囲気」

### カルチャーで例える
- 「まるでSF映画のワンシーンのように」
- 「ビートルズの初期レコードジャケットみたいに」
- 「ジブリ映画の背景画のような」
- 「ナショナルジオグラフィックの表紙のような」
- 「インスタグラムで100万いいねもらえそうな」

## スタイル・画風のキーワード

### アートスタイル
- `水彩画風の優しいタッチ`
- `手書きの版画のような`
- `油絵風の豊かなテクスチャ`
- `ペン画・線画スタイル`
- `コラージュアート風`

### カルチャー・時代
- `1950年代アメリカンレトロ風`
- `1980年代シティポップ風`
- `浮世絵風`
- `バウハウスデザイン風`
- `Y2Kスタイル（2000年代初頭）`

### モダン・デジタル
- `ミニマルデザイン`
- `サイバーパンク`
- `グラスモーフィズム（すりガラス風）`

## 効率化テクニック

### 9分割バリエーション（A/Bテスト向け）
```
以下の要件を満たす広告を9パターン作り、
1枚の画像に9分割のグリッド形式で並べて出力してください。
各パターンはターゲット層や訴求ポイントを少しずつ変えること。
```

### 既存画像を使う（Image to Image）
```
添付画像1枚目をメインビジュアルとして配置し、
全体を[スタイル]になじませて再構成してください。
```

### 英語キーワードの活用
日本語で意図が伝わりにくい場合は英語を混ぜる。
例: `golden hour`, `bokeh background`, `minimalist`, `cinematic lighting`

## アンチパターンと回避策

### 情報過多（最も多い失敗）
- **症状**: 要素が詰め込まれ、何が伝えたいか不明
- **回避**: 「1つのメッセージ」に絞り、構成要素を3〜4つに制限

### ブランドとのズレ
- **症状**: 「高級感」と指定したのに安っぽくなる
- **回避**: カラーパレット・フォント・余白を具体的に指定。生成結果を人間の目で必ず確認

### AI特有の不自然さ
- **症状**: 指が多い、謎の文字、崩れた漢字
- **回避（ネガティブプロンプトに追加）**:
  - `崩れた手`, `不自然な手`, `指が多い`
  - `文字`, `テキスト`, `ロゴ`, `透かし（ウォーターマーク）`
  - 不要な文字が出た場合はMiriCanvasの「AIエリア削除」機能で修正
